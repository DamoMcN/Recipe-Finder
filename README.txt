        
    OVERVIEW
        A simple web-page that generates recipes using 'TheMealDB' API.
        Used to search for recipes or fetch random ones.

    FEATURES
        -Search for recipes by name
        -Displays: 
            recipe image
            category
            area
            ingredients
            instructions
        -Random recipe generator
        -Clean and responsive design
        -Scrollable instructions section

    TECHNOLOGIES USED   
        -HTML5
        -CSS3
        -JavaScript
        -jQuery
        -TheMealDB API

    HOW TO RUN
        1.Download or clone the project.
        2.Open index.html in your browser.
        3.Start searching for recipes.

        No tools or servers required, it all runs on browser.

        Currently optimized for desktop only.
